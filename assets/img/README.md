# Image Assets

현재 작업 환경에서는 외부 이미지 다운로드가 막혀 있어 HTML/CSS에는 기존 Pixabay CDN 주소를 유지했습니다.
운영용 사진을 받으면 이 폴더에 넣고 아래 파일명 기준으로 교체하면 됩니다.

## 권장 파일명

- `hero-conference.jpg` - 첫 화면 배경
- `workshop.jpg` - 소개/워크숍 이미지
- `meeting.jpg` - 세미나/모임 이미지
- `classroom-discussion.jpg` - 성인 학습/토론 이미지
- `teacher.jpg` - 강사/소통 이미지
- `classroom.jpg` - 강의실/프로그램 이미지
